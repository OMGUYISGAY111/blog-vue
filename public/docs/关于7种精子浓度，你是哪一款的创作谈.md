# 测试
## 七种精子和丝之歌
极其痛苦，操纵着小人对着电脑屏幕上的精子视频和100只蟑螂躲了几个小时这事情实在太他妈变态了，但其实也有有趣的地方。
主要是那种未知感和探索感，测试过程中我完全不知道这视频能不能完全被躲开，就有一种作者是我，但对自己的作品完全没有那种掌控感，似乎作品本身还藏了一个无形的作者那种感觉，还挺怪的。
## LNYAAP
这个更痛苦，由于是自己处理视频混剪所以未知感和探索感也没了，变成了纯粹的盲人摸象，唯一算有意思的是每次遇到无解的时候不是改代码而是他妈打开PR给素材移位置再重新导成gif塞里面，十分的漫长，最后因为效果实在太垃圾太不好玩就弃坑了，下面是视频录像
[视频链接](https://www.youtube.com/watch?v=4PVP5JxrdIA)
